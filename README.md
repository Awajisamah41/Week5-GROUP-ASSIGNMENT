# AI Patient Readmission Prediction

## Overview
Predict patient readmission risk within 30 days using AI models.

## Repository Structure
- data/: Raw and processed datasets
- notebooks/: Jupyter notebooks for workflow
- src/: Python modules for preprocessing, modeling, evaluation, deployment
- docs/: Diagrams and supplementary materials

## Setup
1. Install dependencies:


pip install -r requirements.txt


2. Run preprocessing and model scripts or Jupyter notebooks.
